# pubdoc-tools

This repository is a collection of tools for generating, managing, and utilizing
LLM-friendly, version-accurate API documentation for Dart/Flutter packages. It
consists of:

- [dartdoc_builder](dartdoc_builder): A Dart package that provides APIs to
  generate plain-text documentation from Dart source code.
- [pubdoc](pubdoc): A CLI tool for generating and managing version-accurate API
  documentation for Dart/Flutter packages.
- [skill](skill): An agent skill for answering questions about Dart/Flutter
  packages based on API documentation generated by pubdoc.
